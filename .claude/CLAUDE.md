# myApp 项目 — Claude 自动加载规则

> 此文件由 Claude Code 在每次会话开始时自动读取。
> 配置 myApp 项目的个人开发规范和上下文加载。

---

## 快速判断

根据任务关键词，自动加载对应的规范文件：

| 关键词 | 加载文件 | 说明 |
|-------|--------|------|
| 小程序 / wxss / wxml / 微信 | `.ai/quick/小程序开发.md` | 微信小程序开发规范 |
| API / 密钥 / 账户 / 敏感 | `.ai/context/敏感信息.md` | 项目敏感信息参考 |

---

## 完整加载逻辑

```
用户任务
  │
  ├─ 识别关键词
  │   ├─ "小程序开发" / "wxss" / "wxml" → 加载 quick/小程序开发.md
  │   ├─ "API密钥" / "敏感信息" → 加载 context/敏感信息.md
  │   └─ 默认情况 → 仅加载 quick/小程序开发.md
  │
  └─ 开始工作
```

## 手动指令

用户可随时在对话中指定加载规则：

- `「加载小程序规范」` → 加载 `quick/小程序开发.md`
- `「加载敏感信息」` → 加载 `context/敏感信息.md`
- `「轻量模式」` → 仅加载基础规范，不加载敏感信息
- `「完整模式」` → 加载所有可用规范
